# Chrome Dyno AI

Juego simple en el navegador: un cuadrado que salta triángulos. Los triángulos aparecen en pantalla y hay que evitarlos saltando.

**Roadmap:**
- [ ] Juego base (cuadrado + triángulos)
- [ ] Integración de AI que aprende a jugar

## Cómo correr

Abrir `index.html` en el navegador o usar un servidor local:

```bash
# Con Python 3
python3 -m http.server 8000

# O con npx
npx serve .
```
